## <a id="quick-start"></a>Quick Start

### 1. **Install via npm:**

To install package:
```bash
npm install
```

### <a id="setup-provider-credentials"></a>2. **Run Front-End**.
```bash
npm start
```
Runs the app in the browser\
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
